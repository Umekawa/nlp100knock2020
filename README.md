# nlp100knock2020

## これは何
- [自然言語処理100本ノック](https://nlp100.github.io/ja/)を解いてみます。それだけです。
- 章ごとに解いたものをQiitaに整形して上げていきます。 
- 解くのは2020年版です。

## 開発環境
- OS: Windows10
    - 2章のUNIXコマンドはMacで解きます
- Python: たまたま入ってた3.7.7
- エディタ: VSCode
- 他必要があれば追記していく（多分Tensorflowのバージョンとか必要になると思われ）
